# Supply Chain Analytics & Inventory Optimization

A data-driven supply chain analytics project focused on improving demand forecasting, reducing stockouts, and optimizing inventory decisions across multiple warehouses.

---

## 📊 Project Overview
This project analyzes supply chain operations to uncover inefficiencies in demand planning, inventory management, and supplier performance. It combines time series forecasting, inventory analytics, and logistics evaluation to enable smarter, data-driven decision-making.

---

## 🎯 Objectives
- Improve demand forecast accuracy
- Identify and reduce stockouts
- Detect overstock and slow-moving inventory
- Evaluate supplier reliability and delays
- Introduce inventory optimization framework (Safety Stock, ROP, EOQ)

---

## 🧰 Tools & Technologies
- Python (Pandas, NumPy)
- Time Series Forecasting (Prophet, ARIMA)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook
- Excel (baseline comparison)

---

## 🔍 Key Analysis Areas

### 1. Demand Forecasting
- Built time series models to predict demand trends
- Compared ARIMA vs Prophet performance
- Improved forecast accuracy significantly over Excel baseline

### 2. Inventory Analysis
- Identified stockout events using recalculated inventory metrics
- Analyzed stock movement across SKUs and warehouses
- Highlighted imbalance between overstock and understock

### 3. Supplier & Logistics Performance
- Measured delivery delays and supplier reliability
- Flagged high-risk suppliers contributing to disruptions
- Identified patterns in late shipments

### 4. Revenue Forecasting
- Built predictive model for monthly revenue trends
- Aligned demand forecasts with financial projections

---

## 📈 Key Insights
- Forecast accuracy improved significantly, reducing planning errors
- High frequency of stockouts revealed major availability gaps
- Supplier delays identified as a key driver of emergency reorders
- Seasonal demand patterns successfully captured using Prophet
- Bulk order anomalies detected, distorting demand signals

---

## ⚙️ Inventory Optimization Framework
- Safety Stock (SS) model designed for service level control
- Reorder Point (ROP) introduced for proactive replenishment
- Economic Order Quantity (EOQ) framework to balance holding and ordering costs

---

- Reduction in forecast error

- Framework for reducing excess inventory and stockouts
